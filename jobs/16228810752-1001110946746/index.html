Js разработчик

#вакансия #javascript #nodejs #typescript #frontend #middle Компания: #Mетрквадратный https://m2.ru Локация: #москва м.Белорусская Формат работы: офис Занятость: полная З/п: 200к - 260к руб. на руки Контакты: @maxdmitrov Это платформа, на которой физические лица смогут в режиме одного окна подобрать объект недвижимости, оформить на него ипотеку, произвести электронную регистрацию, сделать безопасный расчет не посещая банки и воспользоваться дополнительными услугами (страхование, дизайн, ремонт). Чем предстоит заниматься: • разрабатывать один из продуктов жилищной экосистемы; • прорабатывать задачи с начального этапа (декомпозиция, проработка архитектуры, демо с владельцем продукта); • проводить код ревью; • покрывать код unit тестами, помогать тестировщикам в написании e2e тестов; • оптимизировать рендеринг и скорость загрузки страниц; • разрабатывать продуктовые компоненты, используя общие UI библиотеки, и помогать в развитии этих библиотек. Чего мы ждем: • Javascript(ES6); • React + MobX/Redux; • HTML; • CSS; • Опыт кроссбраузерной верстки. Будет плюсом: • TypeScript; • Знание протокола HTTP/HTTPS, websockets, опыт работы с REST API; • Опыт работы с react native; • Опыт в бекэнде на различных языках программирования; • Опыт профилирования, оптимизации и рефакторинга кода; • Опыт написания тестов. Мы используем: • TypeScript + React + MobX + Gatsby; • NestJS + gRPC + ProtoBuf (мидл-слой, управляющий UI и обращающийся к API бэк-энда); • Micro Frontend Service; • GitLab (CI/CD); • Yarn + Lerna; • Webpack/Rollup. Мы предлагаем: • Современный стек технологий; • Возможность влиять на архитектуру и сразу видеть результат своей работы; • Белая конкурентная ЗП, официальное оформление; • Гибкий график работы, возможность выбора рабочего оборудования; • ДМС, компенсация фитнеса, льготная ставка по ипотеке; • Комфортный офис в БЦ «Белая площадь» (минута от метро): лаундж-зоны, бариста, фрукты.
