Меня зовут Матвей, менеджер проектов в компании The Invaders. Мы занимаемся заказной веб и мобильной разработкой для компаний по всему миру. Основная специализация компании это разработка Web и Mobile проектов, кроме того, в портфеле нашей компании работа с блокчейн проектами на различных платформах (Waves, Ethereum, EOS, Corda, Hyperledger и тд.) Наша команда разработки находится в Новосибирске, но у нас так же, на удаленной основе, работают разработчики с разных мест Земного шара. В данный момент мы ищем колегу DevOps, который поможет нам периодически настраивать, разворачивать сервера, улучшит нашу ситуацию с CI/CD, покажет путь лучшей жизни с контейнеризацией Чем предстоит заниматься: 1. Формирование Docker-образов, CI/CD сценариев 2. Управления контейнеризованными рабочими нагрузками и сервисами (Kubernetes) 3. Решение сетевых вопросов организации серверов 4. Решение вопросов хранения данных 5. Мониторинг систем, сбор, визуализация и анализ логов 6. Взаимодействие с командой разработчиков и совместное решение задач Технические скиллы: 1. Умение работать с Ansible 2. Опыт администрирования баз данных (PostgreSQL, Redis) 3. Опыт работы в качестве Инженера DevOps Технологический стек: - Непосредственное взаимодействие и окружение - Gitlab CI/CD - Docker - Ansible - Ubuntu - PostgreSQL - Redis - Rabbitmq - Python - Django - Celery - Nginx Так же будет очень хорошо: - Увлекаешься программированием - Сможешь рассказать о ревалентном опыте работы Название компании: The Invaders Контакты: hr@theinvaders.pro @matteslip — 👉 Обсуждение вакансии в чате @devops_jobs
